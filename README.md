# 📘 English Janala – Interactive English Learning App

[![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)](https://developer.mozilla.org/en-US/docs/Web/HTML) 
[![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)](https://developer.mozilla.org/en-US/docs/Web/CSS) 
[![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

**English Janala** হলো একটি **interactive, web-based English learning platform**, যা beginner থেকে intermediate learners এর জন্য তৈরি করা হয়েছে।  
এটা vocabulary, sentence examples, mini-lessons, এবং basic grammar কে **visual ও interactive** করে present করে।  

---

## 🎯 Purpose

English শেখা অনেকের জন্য challenge হতে পারে।  
**English Janala** বানানো হয়েছে যাতে English learning:

- সহজ, structured, এবং visual হয়  
- Students ও beginners জন্য user-friendly হয়  
- Daily practice এবং reinforcement সহজ হয়  

---

## ✨ Features

-  **Vocabulary Builder** – key words with meanings  
-  **Sentence Examples** – context-based learning  
-  **Mini Lessons** – structured topic-wise lessons  
-  **Audio Pronunciation** – correct word pronunciation (future)  
-  **Responsive Design** – works smoothly on mobile & desktop  
-  **Fast & Lightweight** – minimal load, clean performance  
-  **Interactive Learning** – engaging UI for better retention  

---

## 📌 How to Use

1. Open the [Live Demo](https://ratul7266.github.io/English_Janala_29/)  
2. Navigate through **lessons / vocabulary / examples**  
3. Practice the words and sentences interactively  
4. Repeat daily for better retention  
5. Future features will allow **audio & quizzes**  

---

## 🛠️ Built With
